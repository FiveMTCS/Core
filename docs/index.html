TCS Core

Presentation

TCS is an open source FiveM framework written in TypeScript.

  • Download the latest version at the releases page
  • Find out how to contribute by reading the contributing guide
  • Report issues here. Please, verify that you are using the latest version first
  • For any question about the core, you can directly ask it into the discussions section

Get started

1. Install tcs_map

  • Download the latest version of tcs_map (it's used to set spawn points)
  • Unzip, then drag and drop it into your FiveM server resources folder

2. Install tcs

  • Download the latest release of tcs
  • Unzip, then drag and drop it into your FiveM server resources folder
  • DO NOT RENAME THIS RESOURCE

3. Complete your server.cfg

Add tcs_map and tcs right after Fivem default resources, it should look like that:

## These resources will start by default.
ensure mapmanager
ensure chat
ensure spawnmanager
ensure sessionmanager
ensure basic-gamemode
ensure hardcap

# TCS
ensure tcs_map
ensure tcs
